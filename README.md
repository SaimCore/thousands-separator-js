## Install
```bash
npm i @saimcore/thousands-separator-js
```

## Usage
```javascript
thousandsSeparator(1000); // 1,000
thousandsSeparator(1000, ','); // 1,000
thousandsSeparator(1000, ''); // 1000
thousandsSeparator(2750.50); // 2,750.50
```
